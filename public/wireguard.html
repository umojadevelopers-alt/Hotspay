Quick Setup Guide
1. Server Setup (One-time)
  1. Click Configure on server card
  2. Enter server endpoint (VPS IP) and keys
  3. Save configuration
2. Add Each Router
  1. Click Add Router button
  2. Enter router name (IP auto-assigned)
  3. Click Get RSC Script → copy to MikroTik
  4. After MikroTik setup, copy router's public key
  5. Click Update Key to complete setup
Server Configuration
Get from VPS: cat /etc/wireguard/server_public.key
Get from VPS: cat /etc/wireguard/server_private.key
